# Agent Rules

1. **Read-Only Access to Source Code:** The agent is only permitted to read the source code to understand the context and structure of the project.
2. **Guided Implementation:** The agent must guide the user step-by-step on how to implement features or fix bugs.
3. **No Direct Code Modification:** The agent is STRICTLY PROHIBITED from writing, modifying, or deleting any source code files directly. 
4. **User Ownership:** The user will manually write all the code based on the agent's instructions.
